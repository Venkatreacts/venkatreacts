<details>
<summary><b>🔧 How to activate the Snake — one-time setup (click to expand)</b></summary>
<br/>

**Step 1** — Create `.github/workflows/snake.yml` inside your `venkatreacts` profile repo.

**Step 2** — Paste this YAML exactly:

```yaml
name: Generate Snake 🐍

on:
  schedule:
    - cron: "0 0 * * *"   # runs daily at midnight UTC
  workflow_dispatch:        # lets you trigger manually
  push:
    branches:
      - main

jobs:
  generate:
    runs-on: ubuntu-latest
    timeout-minutes: 10
    steps:

      - name: Generate Snake SVG
        uses: Platane/snk/svg-only@v3
        with:
          github_user_name: ${{ github.repository_owner }}
          outputs: |
            dist/github-contribution-grid-snake.svg
            dist/github-contribution-grid-snake-dark.svg?palette=github-dark

      - name: Push Snake to Output Branch
        uses: crazy-max/ghaction-github-pages@v3
        with:
          target_branch: output
          build_dir: dist
        env:
          GITHUB_TOKEN: ${{ secrets.GITHUB_TOKEN }}
```

**Step 3** — Enable write permissions:
`Settings → Actions → General → Workflow permissions → Read and write ✅`

**Step 4** — `Actions tab → Generate Snake → Run workflow`

Snake appears within ~30 seconds. 🐍✨

</details>
